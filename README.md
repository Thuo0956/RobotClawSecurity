The Robot Claw Security Agency Website is a professional business website designed to present the company’s security services, build credibility, and provide a direct communication channel between clients and the agency. The website highlights the company’s mission, vision, services, and contact information in a clean and structured layout.

This project is built using standard web technologies and is optimized for clarity, responsiveness, and ease of navigation.

2. Features

Homepage (Home.html)

Company name and branding

Professional navigation bar

Background styling (gradient or video)

Brief company description

Mission and Vision statements

Services Page (Services.html)

Overview of security services offered

Clear explanation of service categories

Structured layout for readability

Contact Page (Contact.html)

Contact form for client inquiries

Company contact details

User-friendly interface

Navigation System

Fully linked navigation menu

Smooth transition between pages

Modern Design Elements

Gradient background (linear-gradient)

Optional background video

Responsive layout structure

Professional typography and spacing

3. Technologies Used

HTML5 – Structure of the website

CSS3 – Styling, layout, gradients, and responsiveness
